# 愛媛マンダリンパイレーツ WordPressテンプレート

WordPressの「カスタムHTML」ブロックに貼り付けて使う、デザイン済みの試合情報ページ一式。
明るいオレンジ基調の「かっこかわいい」トーン（丸ゴシック=M PLUS Rounded 1c＋Oswald、ピル型バッジ、イエローのティッカー）。
演出：回転する集中線・紙吹雪・動くストライプ・光のパルス・浮遊する⚾🍊・日付シャイン・
電光掲示板風ティッカー・ポスターの光るグラデ枠・ボタンのシャイン・マスコットのぴょこぴょこ。
ビルダーのフォーム側はシンプルなUI。

## 主な機能（ビルダー）

- **マスコット2体はチーム名の上に配置**。吹き出しは左右どちらに付けるか選択可（既定は左＝愛媛のキャラ側）
- **コンテンツセクションは自由に追加・削除・並べ替え・見出し変更**が可能
  （初期セット：オープニングアクト／オープニングセレモニー／来場者プレゼント／スタジアムイベント／マンパイパーク／グッズ情報）
- スタジアムイベントには「3回裏終了後 チアダンス」「5回裏終了後 じゃこ天＆お菓子まき」などの実例入り
- **各項目に画像URLを入れると写真付きカード**に、空欄なら文字だけのカードになる
- セクションの形式は「リスト」（箇条書き）と「カード」（バッジ・説明・画像付き）から選択
- 試合日は特大表示。開場（白）・試合開始（イエロー・最強調）・高校生以下無料（ネイビー）の3色チップ
- YouTube配信バンドはサムネ画像URLを入れると画像付きに（空欄なら文字のみ）
- アクセス欄も球場画像URLを入れると写真付きに（空欄なら文字のみ）
- SNSボタンの並びは Instagram → YouTube → X → LINE

## ファイル一覧

| ファイル | 用途 |
|---|---|
| `builder.html` | **【メイン】穴埋めフォーム式ビルダー**。ダブルクリックで開き、入力するだけで試合情報ページのHTMLを生成 |
| `game-info.html` | 試合情報ページのテンプレート本体（手動編集する場合・デザイン改修の原本） |
| `news-list.html` | お知らせ一覧ページ（予備） |
| `preview.html` | ローカルサーバー用プレビュー（通常は使わない） |

## 毎試合の手順（ビルダー使用）

1. `builder.html` をダブルクリックしてブラウザで開く
2. 左のフォームに入力（右のプレビューに即時反映）
   - **基本情報**：試合日（曜日は自動計算）・対戦相手（英字表記も自動）・球場（坊っちゃん/マドンナは住所自動入力）・開場/開始時刻・チケット
   - **画像・マスコット**：WordPressメディアライブラリに画像をアップ →「URLをコピー」して貼る。空欄の項目は自動で非表示
   - **セレモニー／プレゼント／イベント**：行の追加・✕削除ボタンで増減
3. 右上の **「HTMLをコピー」** を押す
4. WordPressで「カスタムHTML」ブロックを追加して貼り付け → プレビュー確認 → 公開

- 入力内容はブラウザに自動保存され、次回開いたとき前回の内容から始められます
- 「ファイル保存」ボタンで生成HTMLを記録用に保存できます
- SNSリンクは初回に実URLを入れれば以後そのまま使われます

### 「CSSを含める」チェックについて
- **ON（既定）**：デザインごと貼り付け。まずはこれでOK
- **OFF**：WordPressの 外観 → カスタマイズ → 追加CSS に `game-info.html` の `<style>` の中身を
  一度登録した後の運用向け。記事側のコードが短くなり、デザイン修正が全記事に一括反映される

## デザインを変えたい場合

`game-info.html` の `<style>` 内を修正（色はCSS変数 `--mp-orange` など）。
ビルダーの生成デザインも変える場合は `builder.html` 内の `TEMPLATE_CSS` に同じ修正を反映する（2箇所で共通のCSS）。

## 設計メモ

- クラス名は `mp-` / `mpg-` プレフィックスでテーマCSSと衝突しない設計。最大幅860px中央寄せ
- アニメーションはCSSのみ（WordPressで安全に動作）。OSの「視差効果を減らす」設定では自動停止
- スマホは1カラム化・マスコット縮小・時刻チップとマスコットの重なり回避済み
- マスコット画像は背景透過PNG推奨（左右の隅からぴょこぴょこ動く。右側は吹き出し付き）
